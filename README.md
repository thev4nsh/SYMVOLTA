# 🎯 SYMVOLTA

> Precision Defines Rank.

A futuristic competitive Android precision game built with Unity and C#.

SYMVOLTA is a neon cyber-style skill game where players draw shapes with maximum possible precision to climb global leaderboards, unlock prestigious ranks, and compete for perfection.

# 🧠 GAME CONCEPT

SYMVOLTA focuses on:
- precision
- muscle memory
- consistency
- competitive skill progression

Players attempt to draw shapes as accurately as possible within a timed session.

The better the precision:
- the higher the score
- the higher the rank
- the higher the leaderboard placement
